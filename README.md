# TapTitans Optimiser
An android app version of Yatto.me, capable of pulling data from the tap titans save file directly

# Current features
 - Stats
  - Shows info about the save file selected
 - Relic Optimiser
  - Different optimisation methods
  - Calculated values displayed
 - Sequencer
  - Current sequence
